Layout Saga

A collection of layout techniques

  • Layout in position fixed [old browser - @Violo solution]
    • fully compatible with old browser (IE6 included)
    • no use of calc value for operation
    • no use of both top:0; bottom:0;
    • spacing layout model by couple of div and padding
  • Layout in position fixed
    • not compatibile with old browser (IE6-7-8, CH18, FF3.x SA5.x)
    • use of calc value for operation
  • Layout in position absolute
    • not compatibile with old browser (IE6)
    • use of both top:0; bottom:0;
    • nestable layout
